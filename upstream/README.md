# Upstream Submission Materials

These files track a possible contribution of the RFC 8259 core to Mojo's standard library.

- `mojo-feature-request.md` follows Modular's current Mojo feature-request form.
- `mojo-stdlib-json-proposal.md` follows the current `mojo/proposals` process.

They are drafts, not submitted upstream. Modular requires maintainer agreement before a non-trivial pull request, and its AI Tool Use Policy requires the human author to review AI-assisted issues, proposals, and code before submission. An upstream submission must disclose `Assisted-by: AI` and use signed commits.

The proposed standard-library scope intentionally excludes JSON5, JSON Patch, Merge Patch, MessagePack, BSON, and CBOR. Those remain external package features unless Modular separately accepts them.
